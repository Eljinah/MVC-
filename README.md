# NYC Crash Data Analytics

## Project Overview
This project focuses on analyzing **New York City (NYC) motor vehicle crash data** to identify key trends, hotspots, and time-based crash patterns.  
It demonstrates a complete **data engineering and analytics pipeline** using **Snowflake**, **Python**, and **SQL** for handling, transforming, and analyzing large-scale data (2M+ records).

Through this project, I showcased my ability to:
- Design and manage **data pipelines** and **data warehouses**
- Perform **complex SQL aggregations** and trend analysis
- Create **data-driven insights** and communicate findings through visualization

---

## Technologies Used
1. Programming Language : Python (Pandas)
2. Data Warehouse : Snowflake
3. Querying : SQL (aggregation, window functions)
4. Data Loading : SnowSQL CLI
5. Visualization : Excel, PowerPoint
